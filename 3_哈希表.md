# 哈希表

## 有效的字母异构词

### 哈希表

哈希表不一定非得用函数，可以用计数排序思想。

```c++
class Solution {
public:
    bool isAnagram(string s, string t) {
        int alphabet[26] = {0};
        for(int i = 0; i < s.size(); i++){
            alphabet[s[i] - 'a']++;
        }
        for(int i = 0; i < t.size(); i++){
            if(alphabet[t[i] - 'a'] == 0){
                return false;
            }
            else{
                alphabet[t[i] - 'a']--;
            }
        }
        for(int i = 0 ; i < 26; i++){
            if(alphabet[i] != 0){
                return false;
            }
        }
        return true;
    }
};
```

### 排序

```c++
class Solution {
public:
    bool isAnagram(string s, string t) {
        if(s.size() != t.size()) return false;
        sort(s.begin(), s.end());
        sort(t.begin(), t.end());
        return t == s;
    }
};
```
