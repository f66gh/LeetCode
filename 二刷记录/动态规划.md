# 动态规划

核心要点：用空间换时间

## 模块识别信号

* 求最值

## 一维DP

### 509. 斐波那契数（母题）

- 状态定义：dp[i] 表示第 i 个斐波那契数
- 转移方程：dp[i] = dp[i - 1] + dp[i - 2]
- 初始化：dp[0] = 0, dp[1] = 1
- 遍历顺序：从小到大递推
- 母题特征：
  1. 当前状态依赖前 1~2 个状态
  2. 问题可以按下标线性展开
  3. 有明确边界和递推公式
  4. 可用滚动变量做空间优化

```cpp
class Solution {
public:
    int fib(int n) {
        if(n < 2) return n;
        int prev1 = 0, prev2 = 1;
        for(int i = 2; i <= n; i++){
            int cur = prev1 + prev2;
            prev1 = prev2;
            prev2 = cur;
        }
        return prev2;
    }
};
```

### 70. 爬楼梯

- 状态定义：dp[i] 表示第 i 个台阶能到达的**组合个数**
- 转移方程：dp[i] = dp[i - 1] + dp[i - 2]（走到台阶只能是前一个台阶夸一步和前二个台阶垮两步，所以把这两个解直接加起来）
- 初始化：dp[1] = 1，dp[2] = 2
- 遍历顺序：从小到大递推

```cpp
class Solution {
public:
    int climbStairs(int n) {
        if(n <= 2) return n;
        vector<int> dp(n + 1);
        dp[1] = 1;
        dp[2] = 2;
        for(int i = 3; i <= n; i++){
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
};
```

### 746. 最小花费爬楼梯

#### 方法1：dp[i]是踩上第i阶楼梯的费用

- 状态定义：dp[i]是踩上第i阶楼梯的**最小费用**
- 转移方程：dp[i] = min(dp[i - 2], dp[i - 1]) + cost[i]（当前阶梯费用）
- 初始化：dp[0] = cost[0]，dp[1] = cost[1]
- 遍历顺序：按索引从小到大递推

```cpp
class Solution {
public:
    int minCostClimbingStairs(vector<int>& cost) {
        int n = cost.size();
        vector<int> dp(n);
        dp[0] = cost[0];
        dp[1] = cost[1];
        
        for(int i = 2; i < n; i++){
            dp[i] = min(dp[i - 2], dp[i - 1]) + cost[i];
        }

        return min(dp[n - 1], dp[n - 2]);
    }
};
```

#### 方法2：dp[i]表示到达第 i 级台阶之前的最小花费

- 状态定义：dp[i]表示到达第 i 级台阶之前的**最小花费**
- 转移方程：dp[i] = min(dp[i - 1] + cost[i - 1], dp[i - 2] + cost[i - 2])（当前阶梯费用）
- 初始化：dp[0] = 0，dp[1] = 0
- 遍历顺序：按索引从小到大递推

```cpp
class Solution {
public:
    int minCostClimbingStairs(vector<int>& cost) {
        int n = cost.size();
        vector<int> dp(n + 1);
        dp[0] = 0;
        dp[1] = 0;
        
        for(int i = 2; i <= n; i++){
            dp[i] = min(dp[i - 2] + cost[i - 2], dp[i - 1] + cost[i - 1]);
        }

        return dp[n];
    }
};
```

## 枚举拆分DP

### 343. 整数拆分（母题）

- 状态定义：dp[i] 表示整数 i 在允许不继续拆分的情况下，可得到的最大乘积值
- 转移方程：dp[i] = max(dp[i], dp[j] * dp[i - j]);
- 初始化：dp[2] = 2，dp[3] = 3
- 遍历顺序：按索引从小到大递推
- 时空复杂度：O(n^2)，O(n)
- 本质：将整数 i 拆成两部分，枚举第一刀的位置

```cpp
class Solution {
public:
    int integerBreak(int n) {
        if(n == 2) return 1;
        if(n == 3) return 2;
		// dp[i] 表示整数 i 在允许不继续拆分的情况下，可得到的最大乘积值
        vector<int> dp(n + 1, 0);
        dp[2] = 2;
        dp[3] = 3;

        for(int i = 4; i <= n; i++){
            for(int j = 2; j <= i / 2; j++){
                dp[i] = max(dp[i], dp[j] * dp[i - j]);
            }
        }

        return dp[n];
    }
};
```

### 96. 不同的二叉搜索树

- 状态定义：dp[i] 表示当前节点个数BST数量
- 转移方程：dp[i] += dp[j] * dp[i - j - 1];（左子树BST个数乘右子树BST个数，利用了BST左右子树都是连号的特性）
- 初始化：dp[0] = 1，dp[1] = 1
- 遍历顺序：按索引从小到大递推
- 时空复杂度：O(n^2)，O(n)

```cpp
class Solution {
public:
    int numTrees(int n) {
        vector<int> dp(n + 1, 0);
        dp[0] = 1;
        dp[1] = 1;
        for(int i = 2; i <= n; i++){
            for(int j = 0; j <= i - 1; j++){
                dp[i] += dp[j] * dp[i - j - 1];
            }
        }

        return dp[n];
    }
};
```

## 网格DP

### 62. 不同路径 II（母题）

- 状态定义：dp[i]是第k行i列的可行路径个数
- 转移方程：if(j != 0) dp[j] = dp[j] + dp[j - 1];（我这里直接简化用一维**滚动数组**了）
- 初始化：dp[0] = 1;
- 遍历顺序：双层循环按索引遍历
- 时间复杂度：O(mn)
- 空间复杂度：O(n)

```cpp
class Solution {
public:
    int uniquePathsWithObstacles(vector<vector<int>>& obstacleGrid) {
        if(obstacleGrid[0][0] == 1) return 0;

        int m = obstacleGrid.size();
        int n = obstacleGrid[0].size();

        vector<int> dp(n, 0);
        dp[0] = 1;

        for(int i = 0; i < m; i++){
            for(int j = 0; j < n; j++){
                if(obstacleGrid[i][j] == 1){
                    dp[j] = 0;
                    continue;
                }
                if(j != 0) dp[j] = dp[j] + dp[j - 1];
            }
        }

        return dp[n - 1];
    }
};
```
