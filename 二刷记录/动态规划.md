# 动态规划

核心要点：用空间换时间

## 模块识别信号

* 求最值
* 求方案数
* 大问题可以拆解为若干小问题
* 存在重复子问题
* 状态之间有递推关系
* 题目往往和这些对象有关：下标位置，前缀区间，子序列，背包容量，树上节点状态

## 一维DP

### 509. 斐波那契数（母题）

- 状态定义：dp[i] 表示第 i 个斐波那契数
- 转移方程：dp[i] = dp[i - 1] + dp[i - 2]
- 初始化：dp[0] = 0, dp[1] = 1
- 遍历顺序：从小到大递推
- 母题特征：
  1. 当前状态依赖前 1~2 个状态
  2. 问题可以按下标线性展开
  3. 有明确边界和递推公式
  4. 可用滚动变量做空间优化

```cpp
class Solution {
public:
    int fib(int n) {
        if(n < 2) return n;
        int prev1 = 0, prev2 = 1;
        for(int i = 2; i <= n; i++){
            int cur = prev1 + prev2;
            prev1 = prev2;
            prev2 = cur;
        }
        return prev2;
    }
};
```

### 70. 爬楼梯

- 状态定义：dp[i] 表示第 i 个台阶能到达的组合个数
- 转移方程：dp[i] = dp[i - 1] + dp[i - 2]（走到台阶只能是前一个台阶夸一步和前二个台阶垮两步，所以把这两个解直接加起来）
- 初始化：dp[1] = 1，dp[2] = 2
- 遍历顺序：从小到大递推

```cpp
class Solution {
public:
    int climbStairs(int n) {
        if(n <= 2) return n;
        vector<int> dp(n + 1);
        dp[1] = 1;
        dp[2] = 2;
        for(int i = 3; i <= n; i++){
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
};
```
