## 一维DP

### 509. 斐波那契数（母题）

- 状态定义：dp[i] 表示第 i 个斐波那契数
- 转移方程：dp[i] = dp[i - 1] + dp[i - 2]
- 初始化：dp[0] = 0, dp[1] = 1
- 遍历顺序：从小到大递推
- 母题特征：
  1. 当前状态依赖前 1~2 个状态
  2. 问题可以按下标线性展开
  3. 有明确边界和递推公式
  4. 可用滚动变量做空间优化

```cpp
class Solution {
public:
    int fib(int n) {
        if(n < 2) return n;
        int prev1 = 0, prev2 = 1;
        for(int i = 2; i <= n; i++){
            int cur = prev1 + prev2;
            prev1 = prev2;
            prev2 = cur;
        }
        return prev2;
    }
};
```

### 70. 爬楼梯

- 状态定义：dp[i] 表示第 i 个台阶能到达的**组合个数**
- 转移方程：dp[i] = dp[i - 1] + dp[i - 2]（走到台阶只能是前一个台阶夸一步和前二个台阶垮两步，所以把这两个解直接加起来）
- 初始化：dp[1] = 1，dp[2] = 2
- 遍历顺序：从小到大递推

```cpp
class Solution {
public:
    int climbStairs(int n) {
        if(n <= 2) return n;
        vector<int> dp(n + 1);
        dp[1] = 1;
        dp[2] = 2;
        for(int i = 3; i <= n; i++){
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[n];
    }
};
```

### 746. 最小花费爬楼梯

#### 方法1：dp[i]是踩上第i阶楼梯的费用

- 状态定义：dp[i]是踩上第i阶楼梯的**最小费用**
- 转移方程：dp[i] = min(dp[i - 2], dp[i - 1]) + cost[i]（当前阶梯费用）
- 初始化：dp[0] = cost[0]，dp[1] = cost[1]
- 遍历顺序：按索引从小到大递推

```cpp
class Solution {
public:
    int minCostClimbingStairs(vector<int>& cost) {
        int n = cost.size();
        vector<int> dp(n);
        dp[0] = cost[0];
        dp[1] = cost[1];
        
        for(int i = 2; i < n; i++){
            dp[i] = min(dp[i - 2], dp[i - 1]) + cost[i];
        }

        return min(dp[n - 1], dp[n - 2]);
    }
};
```

#### 方法2：dp[i]表示到达第 i 级台阶之前的最小花费

- 状态定义：dp[i]表示到达第 i 级台阶之前的**最小花费**
- 转移方程：dp[i] = min(dp[i - 1] + cost[i - 1], dp[i - 2] + cost[i - 2])（当前阶梯费用）
- 初始化：dp[0] = 0，dp[1] = 0
- 遍历顺序：按索引从小到大递推

```cpp
class Solution {
public:
    int minCostClimbingStairs(vector<int>& cost) {
        int n = cost.size();
        vector<int> dp(n + 1);
        dp[0] = 0;
        dp[1] = 0;
        
        for(int i = 2; i <= n; i++){
            dp[i] = min(dp[i - 2] + cost[i - 2], dp[i - 1] + cost[i - 1]);
        }

        return dp[n];
    }
};
```

## 枚举拆分DP

### 343. 整数拆分（母题）

- 状态定义：dp[i] 表示整数 i 在允许不继续拆分的情况下，可得到的最大乘积值
- 转移方程：dp[i] = max(dp[i], dp[j] * dp[i - j]);
- 初始化：dp[2] = 2，dp[3] = 3
- 遍历顺序：按索引从小到大递推
- 时空复杂度：O(n^2)，O(n)
- 本质：将整数 i 拆成两部分，枚举第一刀的位置

```cpp
class Solution {
public:
    int integerBreak(int n) {
        if(n == 2) return 1;
        if(n == 3) return 2;
		// dp[i] 表示整数 i 在允许不继续拆分的情况下，可得到的最大乘积值
        vector<int> dp(n + 1, 0);
        dp[2] = 2;
        dp[3] = 3;

        for(int i = 4; i <= n; i++){
            for(int j = 2; j <= i / 2; j++){
                dp[i] = max(dp[i], dp[j] * dp[i - j]);
            }
        }

        return dp[n];
    }
};
```

### 96. 不同的二叉搜索树

- 状态定义：dp[i] 表示当前节点个数BST数量
- 转移方程：dp[i] += dp[j] * dp[i - j - 1];（左子树BST个数乘右子树BST个数，利用了BST左右子树都是连号的特性）
- 初始化：dp[0] = 1，dp[1] = 1
- 遍历顺序：按索引从小到大递推
- 时空复杂度：O(n^2)，O(n)

```cpp
class Solution {
public:
    int numTrees(int n) {
        vector<int> dp(n + 1, 0);
        dp[0] = 1;
        dp[1] = 1;
        for(int i = 2; i <= n; i++){
            for(int j = 0; j <= i - 1; j++){
                dp[i] += dp[j] * dp[i - j - 1];
            }
        }

        return dp[n];
    }
};
```

## 网格DP

### 62. 不同路径 II（母题）

- 状态定义：dp[i]是第k行i列的可行路径个数
- 转移方程：if(j != 0) dp[j] = dp[j] + dp[j - 1];（我这里直接简化用一维**滚动数组**了）
- 初始化：dp[0] = 1;
- 遍历顺序：双层循环按索引遍历
- 时间复杂度：O(mn)
- 空间复杂度：O(n)

```cpp
class Solution {
public:
    int uniquePathsWithObstacles(vector<vector<int>>& obstacleGrid) {
        if(obstacleGrid[0][0] == 1) return 0;

        int m = obstacleGrid.size();
        int n = obstacleGrid[0].size();

        vector<int> dp(n, 0);
        dp[0] = 1;

        for(int i = 0; i < m; i++){
            for(int j = 0; j < n; j++){
                if(obstacleGrid[i][j] == 1){
                    dp[j] = 0;
                    continue;
                }
                if(j != 0) dp[j] = dp[j] + dp[j - 1];
            }
        }

        return dp[n - 1];
    }
};
```

## 0-1背包

所有物品仅能用一次，即只有两个状态，分为能放入背包和放不进去。

### 416. 分割子集（最值母题）

<div style="color: red">为什么要倒序遍历dp？</div>

<div style="color: blue">因为转移方程的本质是二维的：dp[i][j] = dp[i - 1][j] || dp[i - 1][j - num]，其中i表示用了数组前i个数字，j表示是否能用前i个数字凑出j这个数。而由于每次更新第i行时用不到i-2及之前的行，所以只保留了一行dp，如果不倒序遍历j，那么相当于dp[i][j] = dp[i - 1][j] || dp[i][j - num]，用了本次更新的dp值算了后续的dp值，即这个num可能会被用很多次。</div>

<div style="color: brown">即：当涉及背包问题且用一维dp时，0-1背包用倒序遍历dp，而完全背包用正序遍历dp</div>

**方法1：dp[j]是布尔值，表示j是否能被凑出**

- 状态定义：dp[j]是是否能用前几个数凑出j
- 转移方程：dp[j] = dp[j] || dp[j - num];（我这里直接简化用一维**滚动数组**了）
- 初始化：dp[0] = true;
- 遍历顺序：外边遍历数组，里边**倒序**遍历dp
- 时间复杂度：O(nm)
- 空间复杂度：O(m)

```cpp
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        int n = nums.size();
        int sum = 0;
        for(int i = 0; i < n; i++) sum += nums[i];
        if(sum % 2 == 1) return false;
        int target = sum / 2;

        vector<bool> dp(target + 1, false);
        dp[0] = true;

        for(int num : nums){
            for(int j = target; j >= num; j--){
                dp[j] = dp[j] || dp[j - num];
                if(dp[target] == true) return true;
            }
        }

        return false;
    }
};
```

**方法2：dp[j]表示容量为 j 的背包，能装下的最大价值**

转移方程：dp[j] = max(dp[j], dp[j - num] + num);（当前容量能装下的最多物品容积）

```cpp
class Solution {
public:
    bool canPartition(vector<int>& nums) {
        int n = nums.size();
        int sum = 0;
        for(int i = 0; i < n; i++) sum += nums[i];
        if(sum % 2 == 1) return false;
        int target = sum / 2;

        vector<int> dp(target + 1, 0);

        for(int num : nums){
            for(int j = target; j >= num; j--){
                dp[j] = max(dp[j], dp[j - num] + num);
                if(dp[target] == target) return true;
            }
        }

        return false;
    }
};
```

### 1049. 最后一块石头的重量II

题解同上

```cpp
class Solution {
public:
    int lastStoneWeightII(vector<int>& stones) {
        int sum = 0;
        for(int s : stones) sum += s;
        int target = sum / 2;
        vector<int> dp(target + 1, 0);

        for(int s : stones){
            for(int j = target; j >= s; j--){
                dp[j] = max(dp[j], dp[j - s] + s);
            }
        }

        return sum - 2 * dp[target];
    }
};
```

### 494. 目标和（组合母题）

- 状态定义：dp[j]是凑出j的组合个数
- 转移方程：dp[j] = dp[j] + dp[j - num]; 没有用当前数的组合数+用了当前数的组合数
- 初始化：dp[0] = 1; 当背包什么都不装的时候有且仅有一种组合
- 遍历顺序：外边遍历数组，里边**倒序**遍历dp
- 时间复杂度：O(nm)
- 空间复杂度：O(m)

```cpp
class Solution {
public:
    int findTargetSumWays(vector<int>& nums, int target) {
        int n = nums.size();
        int sum = 0;
        for(int i = 0; i < n; i++) sum += nums[i];
        if((sum + target) % 2 == 1) return 0;
        if(sum < abs(target)) return 0;

        int p = (sum + target) / 2;
        vector<int> dp(p + 1, 0);
        dp[0] = 1;

        for(int num : nums){
            for(int j = p; j >= num; j--){
                dp[j] = dp[j] + dp[j - num];
            }
        }

        return dp[p];
    }
};
```

### 474.一和零（最值组合母题）

- 状态定义：dp\[i][j]是尽可能凑出i,j的最多物品个数
- 转移方程：dp\[i][j] = max(1 + dp\[i - one][j - zero], dp\[i][j]) 用了当前物品的组合数和不用当前物品组合数比大小
- 初始化：dp\[0][0] = 0; 当背包什么都不装的时候，也没有物品组合能装进去
- 遍历顺序：外边遍历数组，里边**倒序**遍历dp
- 时间复杂度：O(lnm)
- 空间复杂度：O(mn)

```cpp
class Solution {
public:
    int findMaxForm(vector<string>& strs, int m, int n) {
        int length = strs.size();

        vector<vector<int>> dp(n + 1, vector<int>(m + 1, 0));

        for(int s = 0; s < length; s++){
            int one = 0, zero = 0;
            for(char c : strs[s]){
                if(c == '0') zero++;
                else one++;
            }

            for(int i = n; i >= one; i--){
                for(int j = m; j >= zero; j--){
                    dp[i][j] = max(1 + dp[i - one][j - zero], dp[i][j]);
                }
            }
        }

        return dp[n][m];
    }
};
```

### 518.零钱兑换 （母题）

- 状态定义：dp[j]是凑出j的组合个数
- 转移方程：dp[j] = dp[j] + dp[j - num]; 没有用当前数的组合数+用了当前数的组合数
- 初始化：dp[0] = 1; 当背包什么都不装的时候有且仅有一种组合
- 遍历顺序：外边遍历数组，里边**正序**遍历dp
- 时间复杂度：O(nm)
- 空间复杂度：O(m)

```cpp
class Solution {
public:
    int change(int amount, vector<int>& coins) {
        int n = coins.size();
        vector<unsigned int> dp(amount + 1, 0);
        dp[0] = 1;

        for(int c : coins){
            for(int j = c; j <= amount; j++){
                dp[j] = dp[j] + dp[j - c];
            }
        }

        return dp[amount];
    }
};
```

### 377. 组合总和 IV

* 母题：70. 爬楼梯

- 状态定义：dp[i]是凑出i的组合个数
- 转移方程：dp[i] += dp[i - num]; 没有用当前数的组合数+用了当前数的组合数
- 初始化：dp[0] = 1; 当背包什么都不装的时候有且仅有一种组合
- 遍历顺序：外边遍历**能凑出的数**，里边**正序**遍历**不同的步数**
- 时间复杂度：O(nm)
- 空间复杂度：O(m)

```cpp
class Solution {
public:
    int combinationSum4(vector<int>& nums, int target) {
        // 这道题是半组合半排列问题，母题是爬楼梯问题，把每一次爬的楼梯变为了数组中的值
        vector<unsigned int> dp(target + 1, 0);
        dp[0] = 1;

        // 先遍历当前的能凑出的和
        for(int i = 1; i <= target; i++){
            // 再遍历一次能爬的楼梯
            for(int num : nums){
                if(i >= num) dp[i] += dp[i - num];
            }
        }

        return dp[target];
    }
};
```
